Selamat Datang di Portofolio Saya

Hai, saya Gianne Angely, seorang mahasiswa tahun pertama di program studi Teknik Informatika yang bersemangat dan penuh rasa ingin tahu. Saya memiliki minat besar dalam pengembangan web dan pengembangan game. Perjalanan saya di dunia teknologi dimulai sejak SMP, dan sekarang saya terus mengembangkan keterampilan saya dalam berbagai proyek yang kreatif dan inovatif.

Sekilas Tentang Karya Saya

Meskipun saya baru di bangku kuliah, saya telah mengerjakan beberapa proyek yang memperkuat kemampuan saya dalam pemrograman web dan game, termasuk:

  • Website Sederhana dengan HTML & CSS: Saya telah membangun berbagai website menggunakan HTML dan CSS tanpa bantuan library atau framework, seperti website dengan desain 3D card, dan website ucapan ulang tahun yang interaktif.
  • Game Visual Novel: Saya juga membuat game visual novel dengan cerita interaktif, di mana pemain bisa menjadi bagian dari alur cerita. Game ini dibuat secara mandiri menggunakan Ren'Py.

Keterampilan dan Keahlian

  • Pengembangan Web Front-End: HTML, CSS, JavaScript
  • Pengembangan Game: Ren'Py untuk visual novel
  • Desain Kreatif: Pembuatan video, desain poster

Mengapa Saya Melakukan Ini

Teknologi bagi saya adalah alat yang luar biasa untuk menciptakan solusi kreatif dan inovatif. Setiap proyek yang saya kerjakan adalah bentuk eksplorasi saya terhadap berbagai konsep teknis dan estetika. Saya selalu berusaha untuk menggabungkan kreativitas dengan logika pemrograman dalam setiap proyek yang saya buat.

Mari Berkolaborasi

Sebagai mahasiswa yang terus belajar dan berkembang, saya selalu terbuka untuk bekerja sama dalam berbagai proyek, baik dalam pengembangan web, pengembangan game, maupun bidang teknologi lainnya. Jangan ragu untuk menghubungi saya melalui bagian kontak jika kamu ingin berkolaborasi!
